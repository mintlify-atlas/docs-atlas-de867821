# List Cohorts

Retrieve a list of all cohorts (user segments) in your project. Supports filtering by type, creator, and search term.

## Endpoint

```http
GET /api/projects/:project_id/cohorts/
```

## Path Parameters

<ParamField path="project_id" type="integer" required>
  The ID of the project to retrieve cohorts from
</ParamField>

## Query Parameters

<ParamField query="type" type="string">
  Filter cohorts by type:
  - `static` - Only static cohorts (fixed membership)
  - `dynamic` - Only dynamic cohorts (auto-updating)
</ParamField>

<ParamField query="created_by_id" type="integer">
  Filter cohorts created by a specific user ID
</ParamField>

<ParamField query="search" type="string">
  Search cohorts by name (case-insensitive partial match)
</ParamField>

<ParamField query="hide_behavioral_cohorts" type="boolean" default={false}>
  If true, exclude cohorts with behavioral filters (useful for feature flag selection)
</ParamField>

<ParamField query="limit" type="integer" default={100}>
  Maximum number of results to return
</ParamField>

<ParamField query="offset" type="integer" default={0}>
  Number of results to skip (for pagination)
</ParamField>

## Response Fields

<ResponseField name="results" type="array">
  Array of cohort objects
  
  <ResponseField name="id" type="integer">
    Unique identifier for the cohort
  </ResponseField>
  
  <ResponseField name="name" type="string">
    Name of the cohort
  </ResponseField>
  
  <ResponseField name="description" type="string">
    Description of the cohort
  </ResponseField>
  
  <ResponseField name="filters" type="object">
    Filter criteria defining cohort membership
  </ResponseField>
  
  <ResponseField name="is_static" type="boolean">
    Whether this cohort is static (true) or dynamic (false)
  </ResponseField>
  
  <ResponseField name="count" type="integer">
    Current number of users in the cohort
  </ResponseField>
  
  <ResponseField name="is_calculating" type="boolean">
    Whether the cohort is currently being recalculated
  </ResponseField>
  
  <ResponseField name="created_by" type="object">
    User who created the cohort
  </ResponseField>
  
  <ResponseField name="created_at" type="string">
    ISO 8601 timestamp of creation
  </ResponseField>
  
  <ResponseField name="last_calculation" type="string">
    ISO 8601 timestamp of last calculation
  </ResponseField>
  
  <ResponseField name="cohort_type" type="string">
    Type of cohort (e.g., "realtime")
  </ResponseField>
  
  <ResponseField name="errors_calculating" type="integer">
    Number of calculation errors encountered
  </ResponseField>
  
  <ResponseField name="last_error_message" type="string">
    Human-readable error message if calculation failed
  </ResponseField>
</ResponseField>

<ResponseField name="count" type="integer">
  Total number of cohorts matching the filters
</ResponseField>

<ResponseField name="next" type="string">
  URL for the next page of results (if available)
</ResponseField>

<ResponseField name="previous" type="string">
  URL for the previous page of results (if available)
</ResponseField>

## Examples

### List All Cohorts

Retrieve all cohorts in your project:

<CodeGroup>
```bash cURL
curl -X GET "https://app.posthog.com/api/projects/12345/cohorts/" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import posthog

posthog.project_api_key = 'YOUR_API_KEY'

cohorts = posthog.get('/api/projects/12345/cohorts/')

print(f"Found {cohorts['count']} cohorts")
for cohort in cohorts['results']:
    print(f"- {cohort['name']}: {cohort['count']} users")
```

```javascript JavaScript
const response = await fetch('https://app.posthog.com/api/projects/12345/cohorts/', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const data = await response.json();
console.log(`Found ${data.count} cohorts`);
data.results.forEach(cohort => {
  console.log(`- ${cohort.name}: ${cohort.count} users`);
});
```
</CodeGroup>

### Filter Static Cohorts

Retrieve only static cohorts:

<CodeGroup>
```bash cURL
curl -X GET "https://app.posthog.com/api/projects/12345/cohorts/?type=static" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import posthog

posthog.project_api_key = 'YOUR_API_KEY'

cohorts = posthog.get('/api/projects/12345/cohorts/', params={'type': 'static'})

print(f"Found {len(cohorts['results'])} static cohorts")
```

```javascript JavaScript
const response = await fetch(
  'https://app.posthog.com/api/projects/12345/cohorts/?type=static',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const data = await response.json();
console.log(`Found ${data.results.length} static cohorts`);
```
</CodeGroup>

### Paginate Results

Retrieve cohorts with pagination:

<CodeGroup>
```bash cURL
curl -X GET "https://app.posthog.com/api/projects/12345/cohorts/?limit=10&offset=20" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import posthog

posthog.project_api_key = 'YOUR_API_KEY'

# Get second page (10 results per page)
cohorts = posthog.get('/api/projects/12345/cohorts/', params={
    'limit': 10,
    'offset': 10
})

print(f"Showing {len(cohorts['results'])} of {cohorts['count']} total cohorts")
```

```javascript JavaScript
const response = await fetch(
  'https://app.posthog.com/api/projects/12345/cohorts/?limit=10&offset=20',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const data = await response.json();
console.log(`Showing ${data.results.length} of ${data.count} total cohorts`);
```
</CodeGroup>

## Error Responses

<ResponseField name="401 Unauthorized" type="error">
  Invalid or missing API key
</ResponseField>

<ResponseField name="403 Forbidden" type="error">
  Insufficient permissions to view cohorts
</ResponseField>

<ResponseField name="404 Not Found" type="error">
  Project not found
</ResponseField>

## Notes

- Results are ordered by creation date (newest first)
- Deleted cohorts are excluded from results
- The `count` field reflects the current number of users, updated during recalculation
- `is_calculating` indicates if the cohort is being processed in the background
- Static cohorts do not recalculate automatically
- Use `hide_behavioral_cohorts=true` when selecting cohorts for feature flags
