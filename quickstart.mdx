## Step 3: Capture your first event

Once installed, PostHog automatically captures pageviews and other events. You can also capture custom events:

<CodeGroup>
```javascript JavaScript
posthog.capture('button_clicked', {
  button_name: 'Sign Up',
  location: 'Homepage'
})
```

```python Python
posthog.capture(
  distinct_id='user123',
  event='button_clicked',
  properties={
    'button_name': 'Sign Up',
    'location': 'Homepage'
  }
)
```
</CodeGroup>

## Step 4: View your data

After capturing events, view them in PostHog:

1. Go to **Activity** in your PostHog dashboard to see live events
2. Create your first **Insight** to analyze trends
3. Set up a **Dashboard** to monitor key metrics
